# Inertial_Measurement_unit

## This IMU file contains library for 
 - GY 81 IMU (9DOF)
 - GY 88 IMU (9DOF)
 - MPU6050   (6DOF)
 - Other accelerometer, gyroscope and compass modules

## Library of Filters for IMU
 - Kalman Filter
 - Complimentary Filter (used inside GY88.cpp)
 - Madwick Filter
 - Mahony Filter
